TASK 1- Responsive Landing Page

##  Overview

BuildFirst is a responsive landing page created using HTML5 and CSS3. It showcases a modern dark-themed design with sections such as a navigation bar, hero banner, features, learning steps, project showcase, call-to-action, and footer.

##  Objective

* Build a responsive landing page using HTML and CSS.
* Practice semantic HTML structure.
* Learn Flexbox, Grid, and responsive design.
* Create a clean and modern user interface.

##  Tools Used

* HTML5
* CSS3
* Google Fonts
* CSS Flexbox
* CSS Grid
* CSS Variables
* Media Queries

##  Steps Performed

1. Created the HTML structure.
2. Designed a fixed navigation bar.
3. Built the hero section with buttons and statistics.
4. Added feature cards and learning steps.
5. Created a project showcase section.
6. Designed a call-to-action and footer.
7. Applied responsive design using media queries.

##  Features

* Responsive layout
* Modern dark theme
* Fixed navigation
* Hero section
* Feature and project cards
* Hover effects and CSS transitions
* Mobile-friendly design

##  Outcome

Successfully developed a modern, responsive landing page using only HTML and CSS. The project demonstrates responsive layouts, reusable UI components, and clean front-end design practices without using any external frameworks or JavaScript
