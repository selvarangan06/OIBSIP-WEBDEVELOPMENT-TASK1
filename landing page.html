// what you will learn

Everything a landing page teaches you

One project, a full mental model. These are the skills you lock in before moving on.

⬛
Document structure

Header, main, footer, nav — how a page is organized before a single pixel of style is applied.

⬜
The box model

Margin, padding, border, content. Once you get this, elements stop behaving like strangers.

⏹
Flexbox & Grid

Two layout systems that handle columns, alignment, and spacing without fighting the browser.

🎨
Color & typography

Picking a palette, pairing fonts, setting a type scale — the craft behind readable, good-looking pages.

📱
Responsive design

Media queries, relative units, and mobile-first thinking so the page works on every screen.

✨
CSS transitions

Hover states, smooth fades, button feedback — the small animations that make a page feel alive.
